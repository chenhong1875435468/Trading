# XAUUSD 趋势 + 震荡网格策略 v6.8 中文复盘报告

生成时间：2026-06-04  
回测模型：1Min OHLC  
策略版本：v6.8 range grid bias  
核心目标：提升震荡行情识别能力，并在高质量震荡中使用受控网格策略。

## 本轮改动

这轮新增了独立的震荡网格模块，但不是无限网格。

网格只在以下条件同时满足时启用：

- 市场被识别为震荡行情。
- Choppiness 足够高，效率比足够低。
- 箱体宽度不能过大。
- 上下边界都至少被触碰确认。
- 价格进入箱体边缘区域。
- 高周期 EMA 有明确方向偏向。

最终保留的关键逻辑是：震荡网格要顺高周期偏向。高周期偏多时，只在箱体下半区做 buy grid；高周期偏空时，只在箱体上半区做 sell grid；高周期方向不清楚则不做。

这解决了前几版网格最大的问题：在黄金长期偏多背景下，盲目做 sell grid 会被上涨趋势突破拖出箱体。

## 为什么这版可以保留

v6.8 的网格不是传统的无脑摊平，而是受控的震荡补充：

- 只在高质量震荡里启用。
- 只在箱体边缘做均值回归。
- 最多 2 个网格仓位。
- 高周期方向不清楚时不做。
- 高周期偏多时不做 sell grid，高周期偏空时不做 buy grid。

它解决了前面几版的核心问题：普通双向网格在黄金上容易逆大方向做空，导致小赚多、一次亏损吃掉很多。

## 当前不足

1. Grid 交易太少，说明识别很保守。
2. 但这个保守是必要的，因为放宽后会明显亏损。
3. 最新区间没有 Grid 交易，样本仍然不足。
4. VeryLong2023 的净利润只比 LongTo0604 多一点，说明 2023 年之前/附近行情对策略贡献有限，还需要继续看数据质量和行情阶段。

## 下一步建议

1. 继续优化箱体质量评分，而不是简单放宽网格条件。
2. 给 Grid 增加“箱体斜率”过滤，避免在缓慢单边趋势中误判震荡。
3. 增加“突破后停止网格”的状态，比如价格连续两根 M15 收在箱体外，就暂停该箱体。
4. 后续可以把 1Min OHLC 与 Real Tick 抽样对比，确认 Grid 的进出场在不同喂价模式下不会差异过大。

## 本轮决策

保留 v6.8。

它不是大幅提高盈利的版本，但它让震荡网格从负贡献变成小幅正贡献，同时没有放大最大回撤。下一步重点是提高震荡识别质量，而不是扩大网格频率。
